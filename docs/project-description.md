# Project Description

Our project is a **Library Management System**. It is a full‑stack web application that manages:

- Books
- Members
- Borrow records
- Reports (JuiceReports)

In Milestone 2 we are only doing planning and design. We created the database design (MongoDB collections and ERD), backend API route plan using TypeScript + Express, and frontend wireframes for the main pages (Home, Data Entry, Data Listing, Edit/Update). These documents will guide development in Milestone 3.
