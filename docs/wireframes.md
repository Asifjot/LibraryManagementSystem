# Frontend Wireframes

The frontend designer created wireframes for the four main pages required in the milestone: Home, Data Entry, Data Listing, and Edit / Update.

- **Home Page** – LMS dashboard with navigation and summary cards.
- **Data Entry Page** – form for adding books, members, and borrow records.
- **Data Listing Page** – table with search, filters, and actions.
- **Edit / Update Page** – form for updating existing records.

The images are stored in `docs/wireframes/`.
